## 运行指南

不建议翻墙运行程序，可能造成翻译时间极长

目前适用架构为Windows x64

在解压后点开 `PyStand.exe` 即可使用

## 代码指南

### 所需库

 - PyQt5
 - requests
 - pandas (预装numpy)
 - BeautifulSoup
 - openpyxl

### 关于打包

并未使用pyinstaller，打包产生的文件过大，运行速度极慢

使用了套壳python3.8架构
